# node-api-with-postgres-DB

# API routes
- GET — / | displayHome()
- GET — /users | getUsers()
- GET — /users/:id | getUserById()
- POST — users | createUser()
- PUT — /users/:id | updateUser()
- DELETE — /users/:id | deleteUser()
